# Stripe Integration

Full-stack Stripe integration with React and Node.js.

## Project Structure

```
stripe/
├── backend/          # Node.js backend server
├── frontend/         # React frontend application
└── .gitignore        # Git ignore rules
```

## Backend

Node.js server for handling Stripe payments and API requests.

### Setup

```bash
cd backend
npm install
npm start
```

## Frontend

React application for the user interface.

### Setup

```bash
cd frontend
npm install
npm run dev
```

## Features

- Stripe payment integration
- Full-stack architecture with separate backend and frontend
- Modern React UI with Vite

## Environment Variables

Create `.env` files in both `backend` and `frontend` directories with your Stripe API keys.
